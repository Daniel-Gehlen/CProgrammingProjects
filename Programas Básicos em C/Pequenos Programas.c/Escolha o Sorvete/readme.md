# Escolha o Sorvete... :

- É o início de um programa que permite ao usuário escolher sorvetes.
- Envolve um menu de opções para sabores de sorvete.

## Fluxograma do Código

``` mermaid
graph TD
    subgraph Main
        M1[Declaração da variável i]
        M2[Loop do-while]
        M3[Switch para escolher o sabor]
        M4[Retorno 0]
    end

    M1 --> M2
    M2 --> M3
    M3 --> M4
```
